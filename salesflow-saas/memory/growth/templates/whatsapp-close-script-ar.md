# WhatsApp Close Script (Arabic) — Dealix

**Goal**: تحويل الفرصة المؤهلة إلى دفعة أولى خلال 24–72 ساعة بعد المكالمة.  
**Tone**: مهني، مباشر، سعودي.

## 1) قواعد التنفيذ
- رسالة واحدة واضحة في كل خطوة.
- لا ترسل أكثر من متابعة واحدة كل 24 ساعة.
- كل متابعة يجب أن تضيف قيمة أو إزالة اعتراض.
- CTA موحّد: "هل نفعّل حسابكم اليوم؟"

## 2) Script Flow

### Message 1 — بعد المكالمة مباشرة (0–30 دقيقة)
مرحبًا {{الاسم}} 👋
سعدت جدًا بمكالمتنا اليوم.
هذا ملخص سريع لما اتفقنا عليه:
- التحدي الحالي: {{pain}}
- الهدف خلال 30 يوم: {{target}}
- الحل المقترح عبر Dealix: {{solution}}

إذا مناسب، أرسل لك الآن العرض المختصر + رابط التفعيل.

### Message 2 — إرسال العرض + الإطار الزمني
تم إرسال العرض ✅
العرض مصمم لفريقكم ويشمل بداية تشغيل خلال {{start_date}}.
الخطوة التالية فقط تأكيد البدء، وبعدها نفعّل الحساب بنفس اليوم.

### Message 3 — اعتراض السعر
أتفهم نقطة الميزانية 👍
عشان تكون الصورة واضحة: الهدف ليس أداة إضافية، بل تقليل ضياع الفرص ورفع الإغلاق.
نقدر نبدأ بباقة {{plan_name}} الآن، ثم نرفعها لاحقًا عند تحقق النتائج.

### Message 4 — اعتراض "مو وقته الآن"
ممتاز، خلنا نخلي البداية خفيفة جدًا:
- تفعيل سريع اليوم
- إعداد أساسي خلال 45 دقيقة
- تشغيل تجريبي أسبوعين
بهذا نأخذ قرارنا على أرقام فعلية، مو توقعات.

### Message 5 — تذكير قبل انتهاء العرض
تذكير بسيط 🙏
عرض التفعيل الحالي ينتهي {{expiry_date}}.
إذا مناسبكم نثبت الاشتراك الآن ونبدأ التنفيذ مباشرة.

### Message 6 — Close Ask (Direct)
{{الاسم}} إذا موافق، أرسل "ابدأ" الآن
وأرسل لك رابط الدفع وخطوات التفعيل خلال دقائق.

## 3) Branching Replies
- إذا قال: "أرسل التفاصيل" → أرسل One-Page Offer PDF + 3 نقاط قيمة فقط.
- إذا قال: "أحتاج شريك يوافق" → أرسل نسخة مختصرة موجهة لصاحب القرار + اطلب موعد 15 دقيقة.
- إذا قال: "ارجع لي لاحقًا" → ثبت تاريخ/وقت محدد للرجوع، لا تتركها مفتوحة.

## 4) No-Response Sequence
- +24h: متابعة قيمة (use case واحدة).
- +48h: متابعة نهائية مع تاريخ انتهاء.
- +72h: إغلاق مؤقت للفرصة مع فتح باب العودة (friendly exit).

## 5) Compliance Notes
- احترام طلبات الإيقاف الفورية.
- عدم مشاركة بيانات حساسة عبر واتساب.
- حفظ جميع المحادثات داخل CRM لأغراض المتابعة والحوكمة.
