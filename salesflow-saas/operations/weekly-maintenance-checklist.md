# قائمة الصيانة الأسبوعية - Dealix

## كل أحد (بداية الأسبوع)
- [ ] مراجعة تقرير الأسبوع السابق
- [ ] التحقق من صحة النسخ الاحتياطية
- [ ] مراجعة أداء المسوقين بالعمولة
- [ ] إرسال تقارير أسبوعية للمسوقين
- [ ] مراجعة العملاء المتوقفين عن الاستخدام

## كل ثلاثاء
- [ ] فحص أمني سريع (logs مشبوهة)
- [ ] مراجعة استخدام الموارد (CPU, RAM, Disk)
- [ ] التحقق من سعة قاعدة البيانات
- [ ] مراجعة أداء الـ Workers
- [ ] تحديث المحتوى التسويقي (إن لزم)

## كل خميس (نهاية الأسبوع)
- [ ] تجميع إحصائيات الأسبوع
- [ ] مراجعة شكاوى الأسبوع ونتائجها
- [ ] تحديث قائمة الأسعار/العروض (إن لزم)
- [ ] مراجعة pipeline المبيعات
- [ ] إعداد ملخص إداري أسبوعي

## شهري (أول أحد من كل شهر)
- [ ] حساب العمولات الشهرية
- [ ] إصدار كشوف العمولات
- [ ] مراجعة شاملة لأداء النظام
- [ ] تحديث المستندات القانونية (إن لزم)
- [ ] مراجعة رضا العملاء
- [ ] تقرير إداري شهري
