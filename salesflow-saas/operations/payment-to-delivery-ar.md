# عملية ما بعد الدفع حتى التسليم - Dealix

## المراحل الرئيسية

### المرحلة 1: استلام الدفع (يوم 0)
- تأكيد الدفع تلقائياً
- إنشاء فاتورة رسمية
- إنشاء سجل عميل في النظام
- إرسال رسالة ترحيب (واتساب + إيميل)
- إسناد مدير حساب

### المرحلة 2: اجتماع الانطلاق (يوم 1-2)
- جدولة اجتماع kickoff (30 دقيقة)
- فهم احتياجات العميل بالتفصيل
- جمع المتطلبات: شعار، بيانات اتصال، قطاع، رسائل مخصصة
- اختيار القالب القطاعي المناسب
- تحديد التوقعات والجدول الزمني

### المرحلة 3: إعداد الحساب (يوم 2-4)
- إنشاء حساب Tenant في المنصة
- تهيئة القالب القطاعي
- إعداد رسائل المتابعة التلقائية
- ربط واتساب بزنس (إذا متوفر)
- إدخال بيانات العملاء الحاليين (إن وجد)

### المرحلة 4: التدريب (يوم 5-7)
- جلسة تدريب أولى (60 دقيقة): نظرة عامة + إدارة العملاء
- جلسة تدريب ثانية (30 دقيقة): التقارير + الأتمتة
- مشاركة فيديوهات تدريبية مسجلة
- فتح قناة دعم مباشرة

### المرحلة 5: الإطلاق التجريبي (يوم 8-14)
- تشغيل المنصة بشكل كامل
- مراقبة الأداء يومياً
- معالجة أي مشاكل فوراً
- تقرير أسبوعي أول

### المرحلة 6: المراجعة والتثبيت (يوم 15-30)
- اجتماع مراجعة بعد أسبوعين
- تعديلات بناءً على الاستخدام الفعلي
- تقرير شهري أول
- تأكيد رضا العميل

### المرحلة 7: الدعم المستمر (بعد يوم 30)
- فترة الضمان الذهبي (30 يوم من الاشتراك)
- دعم فني حسب مستوى الباقة
- تقارير شهرية
- مراجعة ربع سنوية
- عروض ترقية وتحسين

## جدول التسليم

| اليوم | الإجراء | المسؤول |
|-------|---------|---------|
| 0 | تأكيد دفع + ترحيب | النظام |
| 1-2 | اجتماع انطلاق | مدير الحساب |
| 2-4 | إعداد الحساب | الفريق التقني |
| 5-7 | التدريب | مدير الحساب |
| 8-14 | إطلاق تجريبي | مدير الحساب + تقني |
| 15-30 | مراجعة وتثبيت | مدير الحساب |
| 30+ | دعم مستمر | فريق الدعم |

## بوابات الموافقة
1. ✅ العميل يوافق على خطة التنفيذ
2. ✅ العميل يوافق على إعدادات الحساب
3. ✅ العميل يؤكد إكمال التدريب
4. ✅ العميل يوافق على الإطلاق
5. ✅ مراجعة بعد 30 يوم

## مسار البيع الإضافي
- شهر 2: عرض ترقية الباقة
- شهر 3: عرض خدمات إضافية (تكاملات، تخصيص)
- شهر 6: مراجعة شاملة + تجديد
- شهر 12: تجديد سنوي بخصم
