# Metrics Dashboard Kit — Launch to Scale

## North Star Metric
- **Revenue Activated per Active Tenant (monthly)**

## Funnel Metrics
- Leads created
- Qualified leads
- Demos booked
- Proposals sent
- Closed-won
- Stage conversion rates

## Product Usage Metrics
- DAU/WAU by tenant
- Feature adoption rate
- Task completion rate
- Time-to-first-value

## AI Quality Metrics
- AI success rate
- Fallback rate
- Response latency (p50/p95)
- Regeneration rate (proxy for dissatisfaction)

## Sales Metrics
- Win rate
- Average deal size
- Sales cycle duration
- CAC payback estimate

## Operational Metrics
- API error rate
- Queue depth/lag
- Incident count by severity
- MTTR

## Review Cadence
- Daily: health metrics + incidents
- Weekly: funnel + adoption + AI quality
- Monthly: unit economics + strategic bets

## Dashboard Ownership
- Product: adoption + product value
- Engineering: reliability + latency
- Sales: pipeline + conversion
- Leadership: North Star + unit economics
