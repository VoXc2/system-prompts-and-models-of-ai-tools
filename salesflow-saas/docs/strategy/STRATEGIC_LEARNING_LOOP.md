# Dealix — Strategic Learning Loop

## المراجعة الأسبوعية (كل أحد)

| السؤال | الإجراء |
|--------|---------|
| أي رسالة جابت أعلى reply rate؟ | كرّرها |
| أي قناة جابت أكثر demos؟ | ضاعف فيها |
| أي قطاع تجاوب أكثر؟ | ركّز عليه |
| أي رسالة ما جابت نتيجة؟ | أوقفها أو عدّلها |
| أي اعتراض تكرر؟ | جهّز رد أفضل |

## قواعد القرار
```
reply rate < 3% لأسبوعين → غيّر القطاع
demo rate < 20% من الردود → غيّر الرسالة
pilot rate < 30% من demos → غيّر العرض
قناة 0 ردود لأسبوع → أوقفها
رسالة > 15% reply rate → كرّرها everywhere
قطاع > 2 demos/أسبوع → ضاعف الجهد
```

## شروط الانتقال
| من | إلى | الشرط |
|----|-----|-------|
| Manual Penetration | Soft Launch | 10 touches + 1 positive reply |
| Soft Launch | Revenue Live | أول payment مستلم |
| Revenue Live | Repeatable | 3 pilots + 1 sector proven |
| Repeatable | Scale | 10 customers + 30K MRR |
