# Dealix — Agency Negotiation System

## مسار التفاوض مع الوكالات

### A. الافتتاحية
```
"السلام عليكم، شفت أنكم تقدمون خدمات تسويق/دعاية لعملاء.
عندي فكرة تزيد إيرادكم بدون تكلفة: خدمة متابعة leads بعد الحملات.
أنتم تبيعونها، وأنا أشغلها. نبدأ بعميل واحد كتجربة."
```

### B. Discovery (فهم الوكالة)
```
"كم عميل عندكم حالياً؟"
"أي نوع حملات تسوون — performance / سوشال / محتوى؟"
"هل عملاؤكم يشتكون من ضعف التحويل بعد الإعلان؟"
"هل فيه عميل عندكم الحين يخسر leads بسبب بطء الرد؟"
```

### C. تشخيص الألم
```
"المشكلة المشتركة: الوكالة تجيب leads بس ما عندها نظام متابعة بعد الكلك.
العميل يلوم الوكالة. وممكن يطلع.
Dealix يحل هذي المشكلة: يرد على الاستفسارات، يصنفها، يتابع، ويحجز."
```

### D. العرض
```
"النموذج بسيط:
- Setup fee: 3,000 ريال (لكم 1,500)
- الشهري: 990 ريال (لكم 198)
- 10 عملاء = 1,980/شهر + 15,000 setup = 38,760 ريال/سنة
- أنتم تبيعون. أنا أشغل. لا فريق إضافي."
```

### F. الإغلاق
```
"خلينا ما نكبرها. نسوي 3 خطوات:
1. تختار عميل واحد عندك يعاني من ضياع leads
2. نركب flow متابعة بسيط — 7 أيام مجاناً
3. لو اشتغل → نبيعها له كخدمة شهرية

أنت العلاقة. أنا التشغيل. مين أول عميل تبي تجرّب عليه؟"
```
