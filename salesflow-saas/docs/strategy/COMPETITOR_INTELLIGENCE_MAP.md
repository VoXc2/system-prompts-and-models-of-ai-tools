# Dealix — Competitor Intelligence Map

## الملخص
المنافسين العالميين يملكون أجزاء قوية. Dealix يفوز محلياً بجمع الأجزاء في نسخة سعودية: Arabic-first، WhatsApp-first، agency-friendly، human-approved، done-for-you.

---

## A. HubSpot

| البند | التفاصيل |
|-------|---------|
| **وش يسوي** | CRM + Marketing Hub + Sales Hub + Service Hub + Operations Hub |
| **أقوى ميزة** | Workflows، lead scoring، journey orchestration، WhatsApp inside CRM، custom objects |
| **أضعف نقطة** | غالي (500$+/شهر)، معقّد، إنجليزي أولاً، يحتاج أشهر setup، مو مصمم لـ SME سعودي |
| **وش Dealix يقلّد** | Lead scoring بسيط، reply classification، follow-up sequences |
| **وش Dealix يتجنّب** | التعقيد. 100 feature بدل 3 أشياء مركّزة |
| **كيف Dealix يفوز** | أبسط، أرخص (990 vs 500$)، عربي أولاً، done-for-you مو DIY |
| **أولوية** | P1 — نتعلم منه لاحقاً |

**Positioning vs HubSpot:**
> "HubSpot يخزّن بيانات العملاء. Dealix يحرّك العميل للخطوة التالية."
