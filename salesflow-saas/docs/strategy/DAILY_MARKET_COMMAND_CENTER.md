# Dealix — Daily Market Command Center

## القاعدة: 10 touches/يوم minimum

## الروتين

### 8:00 — الصباح (تحضير)
- [ ] شيك كل الردود (email + WA + LinkedIn + X)
- [ ] صنّف كل رد (مهتم/سعر/ديمو/لاحقاً/لا/شريك)
- [ ] احجز أي demo فوراً
- [ ] اختر 10 أهداف اليوم من FIRST_20_TARGETS

### 9:00 — الهجوم
- [ ] LinkedIn: انشر بوست + 5 تعليقات + 3 DMs
- [ ] X: انشر tweet + 5 ردود
- [ ] Email: أرسل 5 إيميلات مستهدفة
- [ ] WhatsApp: 3-5 warm messages

### 14:00 — المتابعة
- [ ] 5 follow-ups (يوم +1/+3/+7)
- [ ] نفّذ أي demo محجوز
- [ ] أرسل payment link لأي مهتم
- [ ] محادثة وكالة واحدة (partner pitch)

### 20:00 — المساء
- [ ] حدّث DAILY_REVENUE_SCORECARD
- [ ] Instagram: story/carousel/reel
- [ ] WhatsApp Status
- [ ] خطّط أهداف بكرا

## قرارات فورية

| الموقف | الإجراء |
|--------|---------|
| أحد قال "مهتم" | احجز demo خلال 24h |
| أحد سأل عن السعر | أرسل pricing + احجز demo |
| أحد قال "مو الحين" | سجّل + follow-up بعد 30 يوم |
| أحد قال "لا" أو "إيقاف" | أوقف فوراً |
| أحد قال "أنا وكالة" | حوّل لـ partner pitch |
| 0 ردود بعد 30 رسالة | غيّر الرسالة أو القطاع |
| ردود بدون demos | غيّر CTA |
| demos بدون دفع | غيّر العرض أو السعر |
