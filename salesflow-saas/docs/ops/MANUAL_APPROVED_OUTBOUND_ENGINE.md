# Dealix — Manual-Approved Outbound Engine

## المبدأ الأساسي
> **AI يبحث ويصيغ ويسجّل. سامي يوافق ويرسل. النظام يتتبّع.**

## المسار
```
AI يبحث → AI يسجّل score → AI يكتب draft → سامي يراجع → سامي يوافق → سامي يرسل → النظام يتتبّع
```

## حدود الإرسال اليومية

| القناة | الحد (أسبوع 1-2) | الحد (بعد إثبات) |
|--------|-----------------|-----------------|
| Email | 10 | 20 |
| WhatsApp warm | 5 | 10 |
| LinkedIn DM | 3 | 5 |
| X reply/DM | 3 | 5 |
| Instagram DM | 2 (inbound only) | 3 |

## Approval Gates
قبل أي إرسال:
- [ ] سامي راجع الرسالة
- [ ] الرسالة مخصصة (مو template عام)
- [ ] فيها opt-out واضح
- [ ] القناة مناسبة للهدف
- [ ] ما تجاوز الحد اليومي

## Status Taxonomy
| الرمز | المعنى |
|-------|--------|
| 🔲 DRAFT | جاهز — ما ترسل |

| 📤 SENT | ترسلت |

| 💰 PAID | دفع |
| ❌ STOPPED | أوقف |
