# Dealix — نظام البيع المباشر (8 قطاعات)

> كل رسالة جاهزة للنسخ والإرسال. عدّل الاسم وأرسل.

---

## 1. عقارات

**الألم:** استفسارات واتساب/إعلانات تبرد بسبب تأخر الرد
**العرض:** Speed-to-Lead Audit + Pilot 499

**أول رسالة:**
```
السلام عليكم، لاحظت أن نشاطكم في العقار يعتمد على الاستفسارات والمتابعة.
Dealix يساعد في ترتيب استفسارات واتساب/إيميل، يفرز الجاد، ويجهز follow-up حتى يتحول الاستفسار لموعد أو عرض.
أقدر أعمل لكم mini audit مجاني: كيف نخلي أول 5 دقائق بعد وصول الـlead أقوى؟
```

**Follow-up +24h:**
```
متابعة بسيطة — لو عندكم leads عقارية كثيرة، أقدر أوريكم flow يفرز الجاد ويجهز ردود متابعة.
```

**Follow-up +72h:**
```
آخر متابعة. إذا يناسبكم أرسل مثال عقاري جاهز للتجربة. إذا لا، أوقف.
```

## 2. عيادات/صحة

**الألم:** حجوزات ضائعة من واتساب
**العرض:** Booking Follow-up Pilot 499

**أول رسالة:**
```
السلام عليكم، عندي ملاحظة بسيطة لقطاع العيادات: كثير استفسارات واتساب ما تتحول لمواعيد لأن المتابعة تتأخر أو تضيع.
Dealix يساعد يفرز الاستفسارات، يجهز ردود متابعة، ويربط المهتم بخطوة حجز واضحة.
ممكن أرسل لكم مثال بسيط على flow لحجز موعد؟
```

**Follow-up +24h:**
```
متابعة فقط — هل عندكم خدمة عليها استفسارات كثيرة وتحتاج متابعة أفضل؟
```

**Follow-up +72h:**
```
آخر متابعة. إذا مناسب أرسل نموذج flow حجز. إذا لا، أوقف.
```

## 3. E-commerce

**الألم:** محادثات واتساب/إنستغرام ما تتحول لطلبات
**العرض:** Inquiry-to-Order Pilot 499

**أول رسالة:**
```
السلام عليكم، المتاجر الإلكترونية تستقبل عشرات الاستفسارات يومياً:
"هل متوفر؟" / "كم التوصيل؟" / "عندكم خصم؟"
كل استفسار ما يُرد عليه = عميل يروح.
Dealix يرد فوراً ويحوّل المحادثة لطلب. مهتم بديمو 10 دقائق؟
```
