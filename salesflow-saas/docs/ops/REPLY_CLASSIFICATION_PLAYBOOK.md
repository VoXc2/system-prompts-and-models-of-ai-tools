# Dealix — Reply Classification Playbook

## 7 تصنيفات + الإجراء الفوري

| التصنيف | أمثلة الردود | الإجراء | السرعة |
|---------|------------|---------|--------|
| **interested** | "مهتم" / "أبي أعرف أكثر" / "حلو" | احجز demo فوراً | خلال ساعة |
| **ask_price** | "كم السعر؟" / "وش التكلفة" / "كم الباقة" | أرسل pricing + احجز demo | خلال ساعة |
| **ask_demo** | "أبي ديمو" / "وريني" / "نشوفه" | احجز فوراً (calendly) | خلال 30 دقيقة |
| **not_now** | "مو الحين" / "لاحقاً" / "مشغولين" | سجّل + follow-up بعد 30 يوم | نفس اليوم |
| **not_fit** | "لا شكراً" / "ما يناسبنا" | أوقف نهائياً + سجّل السبب | فوري |
| **objection** | "غالي" / "عندنا CRM" / "نفكّر" | جاوب من NEGOTIATION_AND_CLOSING_OS + follow-up | خلال ساعة |
| **partner** | "أنا وكالة" / "أقدر أبيعه لعملائي" | حوّل لـ agency partner pitch | خلال ساعة |

## Stop Conditions (أوقف فوراً)
- "إيقاف" / "stop" / "لا" / "لا شكراً"
- ثاني رسالة بدون رد بعد follow-up 2
- أي إشارة عدم اهتمام واضحة
