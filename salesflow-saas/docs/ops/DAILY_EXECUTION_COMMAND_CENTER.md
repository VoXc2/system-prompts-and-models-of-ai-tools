## وش تسوي لما...

### ما فيه ردود
1. غيّر subject line
2. غيّر أول سطر
3. جرّب قناة ثانية (email → LinkedIn)
4. جرّب قطاع ثاني
5. زد عدد الرسائل من 10 لـ 15

### أحد أبدى اهتمام
1. رد خلال ساعة max
2. احجز demo (calendly أو مباشرة)
3. جهّز عرض مخصص بالقطاع
4. أرسل تأكيد الموعد

### بعد demo ناجح
1. أرسل ملخص المحادثة
2. أرسل pilot offer + checkout link
3. Follow-up بعد 24h لو ما دفع
4. Follow-up بعد 72h آخر مرة

### بعد الدفع
1. أرسل رسالة onboarding
2. اطلب: واتساب/إيميل + 3 أسئلة تأهيل
3. فعّل النظام خلال 24 ساعة
4. أرسل تقرير يومي خلال الـ pilot
5. يوم 6: "النتائج + نكمّل بـ Starter؟"
