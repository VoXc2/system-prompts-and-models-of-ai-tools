# Dealix — Service Exchange Playbook

## متى تستخدم التبادل
- وكالة صغيرة أو فريلانسر ما يقدر يدفع
- شريك عنده خدمة تفيد Dealix (تسويق/محتوى/تصميم)
- مرحلة بناء العلاقة قبل الشراكة الرسمية

## الهيكل
```
Dealix يقدم: pilot مجاني + setup لعميل واحد
الشريك يقدم: [خدمة محددة] لمدة 30 يوم
```

## أمثلة تبادل

| الشريك يقدم | Dealix يقدم | المدة |
|------------|-----------|-------|
| 4 بوستات سوشال عن Dealix | pilot + setup لعميلين | 30 يوم |
| 3 إحالات warm | pilot مجاني للشريك | 30 يوم |
| تصميم 2 carousel | pilot + co-selling لعميل | 30 يوم |
| فيديو testimonial | setup مجاني + أولوية دعم | 30 يوم |

## القواعد
1. **30 يوم فقط** — لا خدمة مفتوحة بدون نهاية
2. **نطاق محدد كتابياً** — وش بالضبط كل طرف يقدم
3. **لا تصعيد بدون اتفاق جديد**
4. **التقييم بعد 30 يوم:** نكمّل بنموذج مدفوع أو نوقف

## رسالة العرض
```
"عندي فكرة: أسوي لك النظام مقابل ترويج/محتوى/إحالات.
نحددها 30 يوم بنطاق واضح. لو اشتغلت نكمّل بنموذج ثابت."
```
