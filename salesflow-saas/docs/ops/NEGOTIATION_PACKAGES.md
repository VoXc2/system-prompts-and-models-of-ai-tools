# Dealix — Agency Partnership & Negotiation Packages

> **ابدأ مرن. خلّ الشروط بسيطة. ادفع لأول pilot أو أول إحالة.**

---

## 5 باقات تفاوض

### 1. Referral Partner (أبسط)
| | التفاصيل |
|---|---------|
| **الأفضل لـ** | وكالات عندها عملاء بس ما تبي تشغّل |
| **Dealix يقدم** | Demo + setup + دعم + تنفيذ |
| **الشريك يقدم** | تعريف/إحالة warm |
| **الشروط** | 10-20% referral من أول فاتورة مدفوعة أو مبلغ ثابت |
| **الافتتاحية** | "عطني عميل واحد يعاني من ضياع leads ونثبتها." |
| **الحد** | لا شغل مجاني بدون حد |
| **الإغلاق** | "نبدأ بإحالة واحدة فقط بدون التزام طويل." |

### 2. Implementation Partner (أقوى)
| | التفاصيل |
|---|---------|
| **الأفضل لـ** | وكالات تبي تبيعه كخدمة |
| **Dealix يقدم** | Playbook + setup templates + دعم تشغيلي |
| **الشريك يقدم** | يبيع الباقة للعميل ويدير العلاقة |
| **الشروط** | رسوم Dealix + هامش الوكالة |
| **الافتتاحية** | "بيعوها كخدمة متابعة بعد الحملة." |
| **الحد** | ما يوعد بأتمتة كاملة |
| **الإغلاق** | "أنتم تبيعونها، ونحن نشغلها." |

### 3. Service Exchange (مرحلة البداية)
| | التفاصيل |
|---|---------|
| **الأفضل لـ** | وكالات/فريلانسرز في البداية |
| **Dealix يقدم** | Pilot كامل لـ funnel الوكالة نفسها |
| **الشريك يقدم** | تسويق/ترويج/محتوى/إحالات |
| **الشروط** | تبادل 30 يوم محدد بالنطاق |
| **الافتتاحية** | "أسوي لكم النظام مقابل ترويج/محتوى/إحالات." |
| **الحد** | 30 يوم فقط. لا خدمة مجانية بدون نهاية |
| **الإغلاق** | "نقفلها كتجربة 30 يوم بنتائج واضحة." |

### 4. Co-selling Pilot (عميل مشترك)
| | التفاصيل |
|---|---------|
| **الأفضل لـ** | وكالة عندها عميل نشط |
| **Dealix يقدم** | Audit مشترك + تنفيذ pilot |
| **الشريك يقدم** | وصول للعميل + سياق الحملة |
| **الشروط** | تقسيم setup fee أو عمولة |
| **الافتتاحية** | "نختار عميل واحد ونقفل مشكلة المتابعة." |
| **الحد** | لا نلمس بيانات العميل بدون موافقة |
| **الإغلاق** | "لو ما فيه أثر خلال أسبوع، نوقف." |

### 5. White-label (لاحقاً — P2)
| | التفاصيل |
|---|---------|
| **الأفضل لـ** | وكالات كبيرة |
| **Dealix يقدم** | خيار branding لاحقاً |
| **الشريك يقدم** | volume + عملاء متكررين |
| **الشروط** | ليس الآن — P2 بعد إثبات |
| **الافتتاحية** | "ممكن لاحقاً، الآن نثبت الخدمة باسم Dealix." |
| **الحد** | لا white-label قبل 3 pilots مدفوعة |
| **الإغلاق** | "نثبتها أولاً، بعدها نتفاوض white-label." |

## أقوى مدخل تفاوض

> "نقدر نضيف لكم خدمة تبيعونها لعملائكم: متابعة الـleads بعد الحملات من واتساب/إيميل/نماذج إلى حجز/عرض/تقرير. أنتم تجيبون العميل أو تسوقونها، وأنا أشغل النظام ونبدأ بعميل واحد كتجربة."
