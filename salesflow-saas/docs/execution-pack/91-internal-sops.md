# Internal SOPs

## Daily Operator SOP
**Time**: 09:00 - 09:30 (Arabia Standard Time)
1. Check operator dashboard for overnight issues
2. Review SLA breaches — resolve or escalate
3. Process approval queue (social comments, proposals)
4. Check integration health — fix any failures
5. Review new leads across managed clients — ensure routing
6. Flag anything unusual in Slack/internal channel

## Weekly Growth SOP (Sunday)
1. Review lead volume trends per channel
2. Analyze content performance (engagement, leads generated)
3. Review social listening results
4. Plan next week's content calendar
5. Adjust ad spend based on channel ROI
6. Update ABM target lists

## Weekly RevOps SOP (Tuesday)
1. Pipeline review across all managed tenants
2. Stage aging analysis — escalate stale deals
3. Win/loss analysis for closed deals
4. Sequence performance review (open rates, reply rates)
5. SLA compliance report
6. Update forecasts

## Weekly Sales Enablement SOP (Wednesday)
1. Review new objections encountered
2. Update battle cards if needed
3. Prepare case study from recent wins
4. Update demo environment with latest features
5. Prepare training materials for new features

## Monthly Executive SOP (Last Sunday of Month)
1. Generate executive summary (AI-assisted)
2. Pipeline velocity trend analysis
3. Revenue vs forecast comparison
4. Customer health scoring
5. Churn risk identification
6. Expansion opportunity identification
7. AI cost and quality review
8. Product roadmap alignment

## Onboarding SOP (New Client)
### Day 1
- [ ] Create tenant in platform
- [ ] Configure industry and pipeline stages
- [ ] Set up WhatsApp Business API connection
- [ ] Import existing leads (CSV)
- [ ] Configure first sequence

### Day 2-3
- [ ] Team user accounts created
- [ ] Role permissions configured
- [ ] First playbook activated
- [ ] SLA policies created
- [ ] Training session scheduled

### Day 4-7
- [ ] Training delivered (2 hours)
- [ ] First sequence activated
- [ ] First week monitoring
- [ ] Adjust routing rules based on feedback
- [ ] Go-live confirmation

### Day 8-30
- [ ] Weekly check-in calls
- [ ] Optimize sequences based on data
- [ ] Deliver first monthly report
- [ ] Identify expansion opportunities

## Managed Service SOP (Ongoing)
- **Daily**: Process approval queue, check SLAs, monitor sequences
- **Weekly**: Pipeline review call with client (30 min)
- **Monthly**: Performance report delivery + strategy call (60 min)
- **Quarterly**: Business review + expansion discussion (90 min)
