# Onboarding Packs

## Self-Serve SaaS Onboarding

### Checklist (Stored in tenant.settings.onboarding)
```
Step 1: Complete profile ✓
  - Company name, industry, phone, email
  - ~2 minutes

Step 2: Create your pipeline ✓
  - Choose industry template OR create custom stages
  - ~3 minutes

Step 3: Add your first lead ✓
  - Manual add OR CSV import
  - ~5 minutes

Step 4: Connect WhatsApp ✓
  - WhatsApp Business API setup guide
  - ~15 minutes (with provider)

Step 5: Create your first sequence ✓
  - Choose template OR create custom
  - ~5 minutes

Step 6: Invite your team
  - Add users, assign roles
  - ~3 minutes

Step 7: Activate AI agents (optional)
  - Enable qualification, follow-up agents
  - ~2 minutes
```

### Time-to-Value Target: Under 30 minutes

### Onboarding Emails
- Welcome email (immediate)
- "Complete your setup" reminder (Day 1 if incomplete)
- "Your first week tips" (Day 3)
- "How's it going?" check-in (Day 7)
- Trial expiring reminder (Day 11)
- Trial expired / upgrade (Day 14)

## Managed Service Onboarding

### Phase 1: Discovery (Day 1-3)
- [ ] Discovery call (60 min)
- [ ] Audit current revenue process
- [ ] Document pain points and goals
- [ ] Define KPIs and success metrics
- [ ] Sign service agreement

### Phase 2: Setup (Day 4-7)
- [ ] Create tenant with industry configuration
- [ ] Import existing CRM data (if any)
- [ ] Configure pipeline stages
- [ ] Set up WhatsApp Business API
- [ ] Create industry-specific sequences (5+)
- [ ] Configure SLA policies
- [ ] Create form widgets for website

### Phase 3: Training (Day 8-10)
- [ ] Admin training (2 hours)
- [ ] Sales team training (2 hours)
- [ ] Provide training recordings
- [ ] Deliver user guide (Arabic)

### Phase 4: Go-Live (Day 11-14)
- [ ] Activate sequences
- [ ] Enable AI agents
- [ ] Install website widgets
- [ ] First leads flowing through system
- [ ] Daily monitoring by Dealix operator

### Phase 5: Optimization (Day 15-30)
- [ ] Weekly check-in calls
- [ ] Sequence optimization based on data
- [ ] First monthly performance report
- [ ] Identify quick wins
- [ ] Plan Phase 2 expansion

## White-Label Partner Onboarding

### Phase 1: Agreement (Week 1)
- [ ] Partner agreement signed
- [ ] Revenue share terms defined
- [ ] Branding requirements collected
- [ ] Technical setup call

### Phase 2: Configuration (Week 2)
- [ ] Partner tenant created with elevated permissions
- [ ] Branding applied (logo, colors, fonts)
- [ ] Custom domain configured (if applicable)
- [ ] Partner dashboard configured
- [ ] Demo environment created

### Phase 3: Enablement (Week 3)
- [ ] Partner team training (4 hours)
- [ ] Sales process training
- [ ] Client onboarding process walkthrough
- [ ] Co-branded marketing materials provided
- [ ] Support escalation path defined

### Phase 4: First Client (Week 4)
- [ ] Partner creates first sub-tenant
- [ ] Dealix shadows onboarding
- [ ] Quality review of setup
- [ ] Partner certified to operate independently
