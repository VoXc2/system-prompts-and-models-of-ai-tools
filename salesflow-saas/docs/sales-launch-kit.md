# Sales Launch Kit — Dealix

## ICP (Ideal Customer Profile)
- SMBs في السعودية
- قطاعات البداية: عقار، عيادات، تجزئة، تعليم، مقاولات

## Core Value Proposition
- CRM عربي أولاً + ذكاء اصطناعي تشغيلي
- تسريع تحويل leads إلى صفقات
- تقليل عبء المتابعة اليدوية

## Discovery Questions
1. كيف تديرون leads اليوم؟
2. كم نسبة التسرب في كل مرحلة؟
3. ما أكثر خطوة تستهلك وقت الفريق؟
4. هل تحتاجون أتمتة عربية/واتساب؟

## Objection Handling (مختصر)
- **السعر:** اربط السعر بالعائد المتوقع (time saved + conversion uplift).
- **المخاطر:** اشرح خطة onboarding + support + rollback.
- **الجودة:** اعرض demo حي + KPI baseline.

## Pilot Offer Structure
- مدة: 14–30 يوم
- نطاق واضح: فريق واحد + use cases محددة
- معايير نجاح: conversion, response time, adoption

## Sales Funnel KPIs
- MQL → SQL rate
- SQL → Demo booked
- Demo → Proposal
- Proposal → Closed Won
- Sales cycle length

## Weekly Sales Rhythm
- Pipeline review مرتين أسبوعياً
- Deal clinic أسبوعي للصفقات المعقدة
- Win/Loss review نهاية كل أسبوع
