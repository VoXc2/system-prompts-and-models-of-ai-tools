TOP 10 ESSENTIAL SKILLS
═══════════════════════════════════════════════════════════════

1. Agent Browser (Web Browsing)
   - 180,000+ installs — most installed skill
   - Navigate pages, extract content, follow links
   - Essential for any research/web tasks

2. GOG (Google Workspace)
   - Gmail, Calendar, Drive, Contacts, Sheets, Docs
   - Single integration for entire Google ecosystem
   - Top downloaded skill

3. Tavily Search
   - Search tool built specifically for AI agents
   - Fast, reliable web search
   - Best for research and fact-finding

4. Capability Evolver
   - 35K downloads — tops ClawHub charts
   - Agent automatically evolves its own capabilities
   - Self-improving during operation

5. Summarize
   - Top-3 download on ClawHub
   - Condenses articles, videos, podcasts
   - Quick briefings from any content

6. N8N Workflow Automation
   - Connects to N8N instance
   - Spin up/manage complex multi-step workflows
   - Trigger via cron jobs and plain chat

7. Self-Improving Agent
   - Memory optimization
   - Automated prompt self-correction
   - Learns from failure logs

8. Mission Control
   - Part of "Efficiency Trio" (+ Evolver + GOG)
   - Capability evolution + schedule management
   - Office collaboration

9. ElevenLabs Agents (Voice AI)
   - Integrates with 11Labs CLI
   - Gives agent an actual voice
   - Auto-pivots to phone call if text/email fails

10. GitHub Integration
    - Full GitHub workflow management
    - PRs, issues, code review
    - CI/CD monitoring
