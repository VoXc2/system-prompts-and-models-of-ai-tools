# CEO Command Center

رؤية المدير التنفيذي: **٣ قرارات، فرص، مخاطر، موافقات** — عربي، أزرار ≤٣ في بطاقات الواجهة.

## API ذات صلة

- `GET /api/v1/services/ceo/daily-brief` — Service Tower.
- `GET /api/v1/revenue-os/company-os/command-feed/demo` — دمج Command Feed مع بطاقات Company OS.
- `GET /api/v1/operator/bundles` و`POST /api/v1/operator/chat/message` — مشغّل الخدمات.
