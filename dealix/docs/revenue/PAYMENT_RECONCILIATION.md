# Payment Reconciliation

## Daily (5 min)
- [ ] Check Moyasar dashboard for new payments
- [ ] Update internal tracker (Google Sheet)
- [ ] Send welcome emails for new customers

## Weekly
- [ ] Match Moyasar payments to bank deposits
- [ ] Flag any discrepancies
- [ ] Chase failed/pending payments

## Monthly
- [ ] Reconcile with accountant
- [ ] Generate P&L
- [ ] Review refund requests

## Discrepancy protocol
If Moyasar shows paid but bank doesn't:
1. Check payout schedule (T+1)
2. Wait 48h
3. Contact Moyasar support if still missing
