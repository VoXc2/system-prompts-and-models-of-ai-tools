# استراتيجية التسعير — Dealix

## المبدأ

التسعير يعكس **قيمة السياق + التنفيذ الآمن + إثبات العائد**، وليس عدد المستخدمين فقط. طبقة الأداء اختيارية و**تتطلب تعريف تأهيل + نزاعات + عقود** قبل أي فوترة حقيقية.

---

## 1) Founder Operator — مشغّل المؤسس

| الحقل | التفاصيل |
|--------|-----------|
| **السعر** | **299–499 SAR/شهر** في البيتا الخاصة؛ لاحقاً حتى **999 SAR/شهر**. |
| **لمن؟** | مؤسسون مبكرون، فرق صغيرة جداً، مرحلة التحقق من المنتج-السوق. |
| **ماذا تشمل؟** | موجز يومي عربي، فرص استراتيجية (حد شهري)، ذاكرة مشروع/مسار فهرسة، مسودات رسائل بموافقة، جاهزية إطلاق، رادار سوق محدود. |
| **متى يترقى العميل؟** | عندما يحتاج أنابيب أوسع، مستخدمين متعددين، تكاملات، أو proof أسبوعي رسمي → **Growth OS**. |
| **القيمة** | تركيز يومي + تقليل التشتت + مسار آمن للتواصل. |
| **المخاطر** | توقع «إرسال تلقائي» — يُصحح بالتدريب؛ ضغط تكلفة LLM — يُراقب. |
| **شروط التأهيل** | ICP أو استعداد لتعريفه؛ قبول سياسات الموافقة؛ لا قوائم بارد غير مكبوّتة. |

---

## 2) Growth OS — نظام النمو

| الحقل | التفاصيل |
|--------|-----------|
| **السعر** | **2,999 SAR/شهر**. |
| **لمن؟** | شركات B2B صغيرة/متوسطة، فريق مبيعات يعمل واتساب/بريد بشكل يومي. |
| **ماذا تشمل؟** | Revenue Command Center، Market Radar، سقف استكشاف/إثراء (سياسة)، مسودات AI، تدفق موافقة واتساب، مسودات Gmail، مسودات جدولة اجتماعات، proof pack أسبوعي. |
| **متى يترقى؟** | مقاعد متعددة، تحليلات فريق، تكاملات أعمق، churn/expansion → **Scale OS**. |
| **القيمة** | ربط الإشارة بالقرار بالمسودة بالموافقة بالإثبات. |
| **المخاطر** | توسع نطاق دون امتثال — يُمنع بالسياسات والتدقيق. |
| **شروط التأهيل** | سياسات قنوات موقعة؛ مسؤول امتثال معرف؛ بيانات مصدر شرعية. |

---

## 3) Scale OS — نظام التوسّع

| الحقل | التفاصيل |
|--------|-----------|
| **السعر** | **7,999 SAR/شهر**. |
| **لمن؟** | فرق ناضجة، عدة مقاعد، عمليات إيرادات منظمة. |
| **ماذا تشمل؟** | multi-seat، أداء الفريق، إشارات نجاح العميل، churn/expansion، تكاملات، تحليلات متقدمة، API/webhooks. |
| **متى يترقى؟** | متطلبات SSO، PDPL متقدم، نشر خاص → **Enterprise**. |
| **القيمة** | تنسيق الإيرادات عبر الفريق مع حوكمة. |
| **المخاطر** | تعقيد التكامل — يحتاج خطة تنفيذ. |
| **شروط التأهيل** | خريطة أنظمة؛ DPA؛ مراجعة أمن. |

## 4) Performance Layer — طبقة الأداء (اختياري)

| البند | النطاق |
|--------|--------|
| لكل lead مؤهل | **25–75 SAR** |
| لكل اجتماع محجوز (بعد موافقة) | **150–500 SAR** |
| عمولة نجاح على إيراد مغلق | **3–10%** حسب السوق وحجم الصفقة |

**لمن؟** عملاء يريدون ربط الدفع بالنتيجة بعد تعريف تأهيل صارم.  
**المخاطر:** نزاعات التأهيل — يجب عقد + أدلة + SLA مراجعة.  
**لا تُفعّل** في الإنتاج بدون قانوني واضح.

---

## 5) Enterprise — مؤسسات / نشر خاص

**تسعير مخصص.** SSO، PDPL متقدم، تكاملات، نجاح مخصص، ضوابط بيانات، SLA.

---

## 6) Agency Partner — شراكة وكالات

| البند | النطاق |
|--------|--------|
| إعداد | **3,000–25,000 SAR** |
| حصة إيراد | **15–30%** |

Dealix يحتفظ **باشتراك المنصة**؛ الوكالة تبيع التطبيق والتدريب.
