## 🛡️ القواعد الحديدية (Binding)

1. **Reliability > Security > Measurement > Commercial > Speed > Automation > Scale > Flashy**
2. **Feature freeze 14 days**: Operation + Sell + Measure ONLY
3. **Arabic-first** (اللهجة السعودية الخليجية، RTL)
4. **Token efficiency**: CLI > MCP, grep before read, edit > write
5. **NEVER browser for GitHub** — always gh api
