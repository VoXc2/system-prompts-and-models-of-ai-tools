# رحلات العملاء — Dealix

## 1. سامي كأول مستخدم

- **دخول**: Cursor + ريبو + `/daily-brief`.
- **5 دقائق**: قراءة موجز + فرصة واحدة + تخطي/مسودة.
- **تفعيل**: أول مسودة معتمدة أو جدولة مسودة اجتماع.
- **يومي**: الموجز + الرادار.
- **أسبوعي**: proof pack + تقرير جاهزية.
- **تجديد**: لأنه المنتج نفسه.
- **خطر انقطاع**: ضغط وقت — يحتاج تذكير خفيف.
- **توسعة**: ربط بيتا خارجية.

## 2. مؤسس + Personal Operator

- دخول: دعوة بيتا → تسجيل → موجز.
- تفعيل: 10 فرص مؤهلة على الشاشة.
- يومي: قبول/تخطي/مسودة.
- تجديد: اجتماعات + إثبات.
- churn: عدم فتح الموجز — إنذار صحة حساب.

## 3. SME + Growth OS

- تفعيل: ربط قنوات + سياسات واتساب.
- أسبوعي: proof pack للإدارة.

## 4. مدير مبيعات + Command Center

- لقطة قرارات + رادار + امتثال.

## 5. شريك وكالة

- إعداد + تدريب + حصة.

## 6. Enterprise

- نشر خاص + SSO + PDPL.

## 7. مسؤول تقني

- مفاتيح service role، RLS، فهرسة.

## 8. امتثال

- سجلات، قمع، تصدير/حذف.
