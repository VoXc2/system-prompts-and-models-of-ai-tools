# استراتيجية مشغّل العلاقات — أقوى من Boardy لسياق Dealix

## لماذا أقوى هنا

- **سياق سعودي B2B** + **ذاكرة مشروع** + **ذاكرة إيرادات** + **امتثال** + **إثبات شهري**.
- Boardy يبرز في UX الإنترنت؛ Dealix يربط **العلاقة → الفرصة → المسودة → الموافقة → الاجتماع → الإيراد**.

## القدرات

- اقتراح أشخاص، شرح لماذا الآن، مسودة دافئة، ذاكرة علاقة، مسودة اجتماع، متابعة، مهمة تالية، ربط بالإيراد، تعلم من قبول/تخطي، **لا إرسال تلقائي**.

## فئات العلاقات

مستشار، مستثمر، عميل، شريك، مراجع تقني، مشغّل، إعلام، مرشح توظيف، بناء مجتمع.

## مقاييس

مقبول intros، اجتماعات، شراكات، إيراد مؤثر، قرارات استراتيجية.

- «أعطني 5 مؤسسين سعوديين B2B…»
- «خبير Supabase لمراجعة pgvector…»
