# Demo Script (30 min)

Full script: `docs/sales-kit/dealix_demo_script_30min.md`

## Structure
- 0-3 min: Rapport
- 3-10 min: Discovery (customer talks 70%)
- 10-20 min: Demo on their scenario
- 20-25 min: ROI + pricing
- 25-28 min: Pilot offer
- 28-30 min: Next steps

## Golden rule
Customer speaks 60%, you speak 40%.
