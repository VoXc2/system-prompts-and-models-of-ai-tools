# استراتيجية الأنظمة العمودية — Dealix

## الفكرة

كل عمود له **ألم، مشترٍ، إشارات، زاوية رسالة، مقياس ROI، حساسية تسعير، امتثال**.

## التفاصيل البرمجية

- `GET /api/v1/business/verticals`
- `POST /api/v1/business/verticals/recommend`

## عموديات أساسية

عيادات، عقار، لوجستيات، تدريب، وكالات، مطاعم، ضيافة، إنشاءات، B2B SaaS — انظر JSON في الـ API.
