# إشارات رادار السوق — Dealix

## المصدر البرمجي

- `auto_client_acquisition/v3/market_radar.py` — دالة `signal_catalog()` و`GET /api/v1/v3/market-radar/signal-catalog`.

## مبادئ

لكل إشارة: **لماذا تهم**، **زاوية رسالة**، **قطاعات**، **ثقة تجريبية**، **امتثال**.

## قائمة الإشارات (ملخص)

| إشارة | لماذا |
|--------|--------|
| توظيف مبيعات | ضغط أنابيب / توسع |
| فرع جديد | توسع جغرافي |
| موقع محدث | حملة أو منتج |
| رابط حجز | مسار تحويل واضح |
| إعلان جديد | استثمار بالطلب |
| تمويل | نافذة شراء |
| فعالية | شبكة علاقات |
| شراكة | قنوات جديدة |
| إطلاق منتج | تفعيل |
| مناقصة | B2B رسمي |
| تقييمات متغيرة | ضغط تجربة |
| وظائف | نمو تنظيمي |
| أثر CRM | نضج عمليات |
| واتساب ثقيل | قناة لكن بموافقة |
| بطء رد | تسريب فرص |
| حملة منافس | رد ذكي لا تقليد |

**امتثال:** لا واتساب تسويقي بدون opt-in؛ سجل موافقات.
