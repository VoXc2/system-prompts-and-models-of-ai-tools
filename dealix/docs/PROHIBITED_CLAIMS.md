# ادعاءات ممنوعة في التسويق والمبيعات

لا تُستخدم هذه العبارات في صفحات الهبوط، الإعلانات، الرسائل، أو الديمو:

- «نضمن لك عملاء» أو «نضاعف مبيعاتك مضموناً».
- «نرسل لكل السوق» أو «نوصل لكل الأرقام».
- «نسحب بيانات LinkedIn» أو «نستخرج جهات الاتصال تلقائياً من LinkedIn».
- «نرسل واتساب بارد لقائمة كبيرة بدون موافقة».
- «نعمل بدون موافقتك» أو «نُرسل نيابة عنك مباشرة».
- «نتجاوز سياسات المنصات» أو «نتفادى الحظر بطرق غير شرعية».
- أي وعد بنتائج مالية محددة بدون أساس قابل للتحقق والعقد.

## بدائل مسموحة

- «نُجهّز فرصاً مؤهّلة ورسائل عربية، وأنت توافق قبل أي تواصل حساس».
- «نقيّم قابلية التواصل (safe / needs_review / blocked) لتقليل المخاطر».
- «Proof Pack يوثّق المسودات والموافقات والمخاطر التي تم منعها» (حسب ما يُقاس فعلياً).
