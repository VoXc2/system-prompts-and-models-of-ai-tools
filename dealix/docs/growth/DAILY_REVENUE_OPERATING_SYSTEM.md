## 🧪 Weekly Experiments

**Every Friday:** Design 1 experiment for next week.

Examples:
- Try voice notes on WhatsApp (vs text)
- Shorter subject lines on email
- Different time of day for outreach
- New channel (Instagram DM, TikTok)
- New lead source (Product Hunt, niche forums)

**Measure:** compare response rate vs. baseline.
**Rule:** Keep what works, kill what doesn't, within 2 weeks.

---

## 🎯 Conversion Funnel Targets (Daily)

If you're hitting **0 customers/day:**
- Need more **quality** leads, not more volume
- Likely: wrong ICP, weak message, or wrong channel

If you're hitting **0.5-1 customers/day:**
- System works. Need more volume.
- Scale channels that convert best.

If you're hitting **1-3 customers/day:**
- You're in product-market fit zone
- Focus on retention + referrals now
- Consider hiring first BDR

---

## 🔄 Bottleneck Diagnosis

Every week, identify the narrowest funnel point:

| If this is low... | Action |
|-------------------|--------|
| Leads/day | Add source (agency, paid ads) |
| Reply rate | Rewrite opening line |
| Demo booking | Simplify ask (15 not 30 min) |
| Demo show rate | Confirmation 24h before |
| Demo close | Practice objection handler |
| Payment completion | Simplify checkout |

Fix the narrowest point first. Don't add volume upstream if conversion downstream is broken.

## 💪 Mental Game

**Week 1-2:** Expect 0 paid customers. Don't panic.
**Week 3-4:** Expect 1-3 paid. Relief sets in.
**Month 2:** Expect 5-10 paid. Momentum builds.
**Month 3:** Expect 15-30 paid. Real business emerges.

The first customer is the hardest. Everything after is easier because:
- Social proof exists
- You know what works
- You can do referrals
- Your confidence is real, not performed

**Daily discipline > sporadic genius.**

Run this system every day. Results compound.
