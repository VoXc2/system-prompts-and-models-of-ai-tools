# أدلة ربط الموصلات (ملخص)

- **واتساب:** opt-in واضح؛ الإرسال الحي behind flag وسياسة؛ مراجعة قوائم قبل أي حملة.
- **Gmail:** مسودات أولاً (`gmail.compose` لاحقاً عبر OAuth) — لا إرسال من Dealix افتراضياً.
- **Calendar:** مسودات/موافقة قبل `events.insert`.
- **Moyasar:** sandbox أو فاتورة يدوية من لوحة التحكم؛ لا تخزين بيانات بطاقة داخل Dealix في MVP.
- **LinkedIn Lead Gen:** إعلانات + نماذج رسمية؛ لا scraping ولا auto-DM.

**مرجع حالة تجريبية:** `GET /api/v1/customer-ops/connectors/status`
